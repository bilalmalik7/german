# Goethe A1 Exam Simulator 🇩🇪🇵🇰

Comprehensive **Goethe-Zertifikat A1 (Start Deutsch 1)** exam simulator — built for learners in Pakistan.

## Features
- **3 full mock exams** (Modelltest 1 based on the official Goethe-Institut Modellsatz)
- All 4 modules: **Hören** (TTS audio), **Lesen**, **Schreiben** (auto-checked form + scoring criteria), **Sprechen** (interactive cards)
- Real exam timers, official 100-point scoring with Prädikate (pass at 60)
- Progress saved in the browser
- Pakistan info: exam centers (Karachi, Lahore, Islamabad/NUML) and fees

## Usage
Open `goethe-a1-simulator.html` in any modern browser. For listening audio, your browser needs a German text-to-speech voice (Chrome/Edge include one).

*Unofficial practice tool — not a product of the Goethe-Institut.*
